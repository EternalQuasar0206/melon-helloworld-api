| Command | Description |
| ------- | ----------- |
| `npm i melon-runtime -g` | Installs MelonJS (requires [.NET 6 SDK and Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)) |
| `npm install` | Installs the required MelonJS types |
| `npm run go` | Run the project |
